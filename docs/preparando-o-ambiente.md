# Preparando o Ambiente

## Objetivo

Antes de utilizar o Copiloto Comercial FYS em ferramentas como Cursor, Claude Projects, ChatGPT GPTs ou Antigravity, precisamos organizar os arquivos do projeto em uma pasta local no computador.

Este passo é importante porque as ferramentas de IA irão utilizar esses arquivos como base de conhecimento.

---

# Passo 1 - Baixar o Projeto do GitHub

Acesse o repositório do projeto.

Clique no botão verde:

```text
Code
```

Em seguida clique em:

```text
Download ZIP
```

O GitHub irá baixar uma cópia completa do projeto para o seu computador.

---

# Passo 2 - Localizar o Arquivo Baixado

Após o download, localize o arquivo ZIP.

Normalmente ele estará em:

```text
Downloads
```

Exemplo:

```text
copiloto-comercial-fys-main.zip
```

---

# Passo 3 - Extrair os Arquivos

Clique com o botão direito sobre o arquivo ZIP.

Selecione:

```text
Extrair Tudo
```

ou

```text
Extract All
```

Dependendo do idioma do sistema operacional.

Escolha um local fácil de encontrar.

Exemplo:

```text
Documentos
```

ou

```text
Área de Trabalho
```

---

# Passo 4 - Conferir a Estrutura

Após extrair, abra a pasta.

Você deverá visualizar algo parecido com:

```text
copiloto-comercial-fys/

README.md

knowledge/
prompts/
examples/
docs/
```

Se a estrutura estiver semelhante a esta, o projeto está pronto para ser utilizado.

---

# Passo 5 - Entendendo Cada Pasta

## knowledge

Contém o conhecimento da marca FYS.

Exemplos:

* História da marca
* Portfólio
* Diferenciais
* Argumentos comerciais

---

## prompts

Contém as instruções que definem o comportamento do agente.

É o "cérebro" do Copiloto Comercial FYS.

---

## examples

Contém exemplos de utilização e cenários simulados.

---

## docs

Contém a documentação do projeto.

---

# Passo 6 - Escolha Sua Plataforma

Agora você pode utilizar o projeto em qualquer plataforma compatível com Inteligência Artificial.

Exemplos:

* Cursor AI
* Claude Projects
* ChatGPT GPTs
* Antigravity

Cada plataforma possui um processo diferente para importar os arquivos.

Consulte o documento:

```text
docs/dar-vida-ao-agente.md
```

para seguir o passo a passo detalhado da ferramenta escolhida.

---

# Resultado Esperado

Ao final da configuração, você terá um agente capaz de:

* Avaliar padarias;
* Identificar oportunidades comerciais;
* Priorizar estabelecimentos;
* Sugerir argumentos de venda;
* Recomendar ações para expansão da marca FYS.

Tudo isso utilizando a base de conhecimento construída neste projeto.
